## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/haashirk633-design/Jsw-python-projects.git
```

### Open Project Folder

```bash
cd Jsw-python-projects
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Run Python Program

```bash
python filename.py
```
